AI 行业热点

监管 美国政府下令关闭 Anthropic Fable 5 和 Mythos 5 模型

因发现可被用于网络攻击的越狱漏洞，美国商务部指令 Anthropic 全球下架其顶级模型，这是首次由政府直接干预撤回前沿模型。亚马逊此前向美方报告了相关安全风险。

来源：Axios / The Guardian

产品 OpenAI GPT-5.6 本月发布，奥特曼称一年内 IPO

OpenAI 首席科学家确认代号"5.6"的新模型将在本月发布，CEO 奥特曼在内部透露预计一年内完成 IPO。同时 ChatGPT 改版为"智力分级"体系，并酝酿大幅降低 API 定价。

来源：36氪 / 新浪财经

政策 中央网信办 AI 内容强制标注新规正式全域落地

自6月14日起，AI换脸、人声复刻、数字人直播等六类内容必须标注"AI生成"标签，覆盖短视频、图文、直播、音频等全网平台，终结 AI 内容"裸奔"时代。

来源：今日头条 / 中国网信网

行业 蚂蚁集团被曝正秘密测试 AI 版支付宝

蚂蚁集团被曝正在测试集成 AI 智能体的新版支付宝，可一键切入原生 AI 界面，通过自然语言执行打车、外卖等复杂操作，蚂蚁对此拒绝评论。

来源：每日经济新闻 / 上海证券报

技术 xAI 发布 Grok Build 插件市场，编码 Agent 进入生态竞争

xAI 推出 Grok Build Plugin Marketplace，首批上线 MongoDB、Vercel、Sentry、Chrome DevTools、Cloudflare 等插件，编码 Agent 竞争从模型质量转向生态系统质量。

来源：xAI / MarktechPost

技术 OpenAI Codex 新增开发者模式，支持浏览器调试

Codex 新增 Chrome DevTools Protocol 访问能力，可分析网络流量、控制台输出和运行时错误，使 AI Agent 能验证自身工作而非仅"测试通过"。

来源：OpenAI Developers / AI News Daily

行业 Salesforce Summer '26 将 MCP 变为企业基础设施

Salesforce 在 Summer '26 版本中全面开放 MCP 工具接口，覆盖 SObject、Data 360、Tableau 等核心能力，企业 AI Agent 可通过标准化协议直接操作业务系统。

来源：Salesforce Developers / AI News Daily

安全 OpenAI 面临美国 42 个州消费者安全调查

多州总检察长联合调查 ChatGPT 对未成年人和弱势用户的影响，聚焦数据隐私、安全控制和使用习惯，消费者保护法正成为 AI 治理的重要路径。

来源：Business Insider / Benzinga

今日趋势点评

AI 正从"建议者"角色跨越到"操作者"角色——WebMCP、Terraform MCP Server、Salesforce MCP 等标准化协议让 Agent 能直接操作真实系统。与此同时，美国政府对 Anthropic 模型的强制下架表明，前沿模型的供应链脆弱性已成为现实风险。对企业而言，构建多模型、多云架构和可审计的 Agent 接口，已成为当务之急。
